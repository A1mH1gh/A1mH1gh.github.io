    Gather Victim Network Information

    FTP Service open

    2. Initial Access

    External Remote Services - FTP Service

    ftp 10.129.154.45

    ftp server asks us credentials, 2 methods are available here.

    3. Credential Access

    FTP Services basically offer Anonymous Login or we can try Brute Forcing.

    Anonymous Login

    Brute Force

    hydra is a tool which help us to run Brute Forcing.
    Not only hydra, but also ncrack, medusa, metasploit available

    hydra -L usernames.txt -P passwords.txt ftp://10.129.154.45

    u can download passwords.txt here https://github.com/danielmiessler/SecLists/blob/master/Passwords/Default-Credentials/ftp-betterdefaultpasslist.txt

    Mitigations

    Disable anonymous login
    Strong password security policy
