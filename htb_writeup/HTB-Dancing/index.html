    1. RECONNAISSANCE

    Gather Victim Network Information

    nmap -sS -sV 10.129.1.12

    445 port is open, SMB(Server Message Block) Protocol which provides shared access to files, Directory, printers on Windows.

    crackmapexec smb 10.129.1.12 --shares

    we also get OS Information via crackmapexec tool 

    smbclient -L \\10.129.1.12

    4 shares are here, But only WorkShares accessible 

    smbclient \\\\10.129.1.12\\WorkShares



    Done.

    Mitigations

    Make sure Password Policies are proper
    Refer to NIST guidelines when creating password policies.
